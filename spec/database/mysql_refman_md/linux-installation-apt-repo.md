### 2.5.2 Installing MySQL on Linux Using the MySQL APT Repository

The MySQL APT repository provides `deb`
packages for installing and managing the MySQL server, client, and
other components on the current Debian and Ubuntu releases.

Instructions for using the MySQL APT Repository are available in
[A Quick
Guide to Using the MySQL APT Repository](https://dev.mysql.com/doc/mysql-apt-repo-quick-guide/en/).
