#### 15.7.7.19 SHOW FUNCTION CODE Statement

```sql
SHOW FUNCTION CODE func_name
```

This statement is similar to [`SHOW PROCEDURE
CODE`](show-procedure-code.md "15.7.7.27 SHOW PROCEDURE CODE Statement") but for stored functions. See
[Section 15.7.7.27, “SHOW PROCEDURE CODE Statement”](show-procedure-code.md "15.7.7.27 SHOW PROCEDURE CODE Statement").
