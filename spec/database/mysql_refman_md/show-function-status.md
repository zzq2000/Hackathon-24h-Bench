#### 15.7.7.20 SHOW FUNCTION STATUS Statement

```sql
SHOW FUNCTION STATUS
    [LIKE 'pattern' | WHERE expr]
```

This statement is similar to [`SHOW PROCEDURE
STATUS`](show-procedure-status.md "15.7.7.28 SHOW PROCEDURE STATUS Statement") but for stored functions. See
[Section 15.7.7.28, “SHOW PROCEDURE STATUS Statement”](show-procedure-status.md "15.7.7.28 SHOW PROCEDURE STATUS Statement").
