| **A.8.1.** | Where can I find information on how to upgrade or downgrade MySQL? |
|  | For detailed upgrade information, see [Chapter 3, *Upgrading MySQL*](upgrading.md "Chapter 3 Upgrading MySQL"). Do not skip a major version when upgrading, but rather complete the process in steps, upgrading from one major version to the next in each step. This may seem more complicated, but ultimately saves time and trouble. If you encounter problems during the upgrade, their origin is easier to identify, either by you or, if you have a MySQL Enterprise subscription, by MySQL support. |
