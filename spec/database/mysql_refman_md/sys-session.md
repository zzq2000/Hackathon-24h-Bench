#### 30.4.3.33 The session and x$session Views

These views are similar to
[`processlist`](sys-processlist.md "30.4.3.22 The processlist and x$processlist Views") and
[`x$processlist`](sys-processlist.md "30.4.3.22 The processlist and x$processlist Views"), but they filter
out background processes to display only user sessions. For
descriptions of the columns, see
[Section 30.4.3.22, “The processlist and x$processlist Views”](sys-processlist.md "30.4.3.22 The processlist and x$processlist Views").
