#### 30.4.3.20 The memory\_global\_total and x$memory\_global\_total Views

These views summarize total memory use within the server.

The [`memory_global_total`](sys-memory-global-total.md "30.4.3.20 The memory_global_total and x$memory_global_total Views") and
[`x$memory_global_total`](sys-memory-global-total.md "30.4.3.20 The memory_global_total and x$memory_global_total Views") views have
these columns:

- `total_allocated`

  The total bytes of memory allocated within the server.
