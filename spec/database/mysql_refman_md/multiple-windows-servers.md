### 7.8.2 Running Multiple MySQL Instances on Windows

[7.8.2.1 Starting Multiple MySQL Instances at the Windows Command Line](multiple-windows-command-line-servers.md)

[7.8.2.2 Starting Multiple MySQL Instances as Windows Services](multiple-windows-services.md)

You can run multiple servers on Windows by starting them manually
from the command line, each with appropriate operating parameters,
or by installing several servers as Windows services and running
them that way. General instructions for running MySQL from the
command line or as a service are given in
[Section 2.3, “Installing MySQL on Microsoft Windows”](windows-installation.md "2.3 Installing MySQL on Microsoft Windows"). The following sections
describe how to start each server with different values for those
options that must be unique per server, such as the data
directory. These options are listed in
[Section 7.8, “Running Multiple MySQL Instances on One Machine”](multiple-servers.md "7.8 Running Multiple MySQL Instances on One Machine").
