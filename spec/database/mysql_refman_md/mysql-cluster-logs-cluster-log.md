#### 25.6.2.1 NDB Cluster: Messages in the Cluster Log

The following table lists the most common
[`NDB`](mysql-cluster.md "Chapter 25 MySQL NDB Cluster 8.0") cluster log messages. For
information about the cluster log, log events, and event types,
see [Section 25.6.3, “Event Reports Generated in NDB Cluster”](mysql-cluster-event-reports.md "25.6.3 Event Reports Generated in NDB Cluster"). These log
messages also correspond to log event types in the MGM API; see
[The Ndb\_logevent\_type Type](https://dev.mysql.com/doc/ndbapi/en/mgm-types.html#mgm-ndb-logevent-type), for related information
of interest to Cluster API developers.
