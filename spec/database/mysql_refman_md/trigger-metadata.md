### 27.3.2 Trigger Metadata

To obtain metadata about triggers:

- Query the [`TRIGGERS`](information-schema-triggers-table.md "28.3.45 The INFORMATION_SCHEMA TRIGGERS Table") table of the
  `INFORMATION_SCHEMA` database. See
  [Section 28.3.45, “The INFORMATION\_SCHEMA TRIGGERS Table”](information-schema-triggers-table.md "28.3.45 The INFORMATION_SCHEMA TRIGGERS Table").
- Use the [`SHOW CREATE TRIGGER`](show-create-trigger.md "15.7.7.11 SHOW CREATE TRIGGER Statement")
  statement. See [Section 15.7.7.11, “SHOW CREATE TRIGGER Statement”](show-create-trigger.md "15.7.7.11 SHOW CREATE TRIGGER Statement").
- Use the [`SHOW TRIGGERS`](show-triggers.md "15.7.7.40 SHOW TRIGGERS Statement")
  statement. See [Section 15.7.7.40, “SHOW TRIGGERS Statement”](show-triggers.md "15.7.7.40 SHOW TRIGGERS Statement").
