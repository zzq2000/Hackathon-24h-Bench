#### 15.7.7.8 SHOW CREATE FUNCTION Statement

```sql
SHOW CREATE FUNCTION func_name
```

This statement is similar to [`SHOW CREATE
PROCEDURE`](show-create-procedure.md "15.7.7.9 SHOW CREATE PROCEDURE Statement") but for stored functions. See
[Section 15.7.7.9, “SHOW CREATE PROCEDURE Statement”](show-create-procedure.md "15.7.7.9 SHOW CREATE PROCEDURE Statement").
