#### 15.6.5.3 ITERATE Statement

```sql
ITERATE label
```

[`ITERATE`](iterate.md "15.6.5.3 ITERATE Statement") can appear only within
[`LOOP`](loop.md "15.6.5.5 LOOP Statement"),
[`REPEAT`](repeat.md "15.6.5.6 REPEAT Statement"), and
[`WHILE`](while.md "15.6.5.8 WHILE Statement") statements.
[`ITERATE`](iterate.md "15.6.5.3 ITERATE Statement") means “start the
loop again.”

For an example, see [Section 15.6.5.5, “LOOP Statement”](loop.md "15.6.5.5 LOOP Statement").
